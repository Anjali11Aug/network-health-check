# Network Infrastructure Automation & Monitoring Tool

Python-based network automation project that monitors device health and latency using automated ping checks and displays results on a real-time Flask dashboard.

## Features
- YAML-based device configuration
- Automated health checks
- Latency measurement
- JSON logging
- Web dashboard
- Auto refresh

## Tech Stack
Python, Flask, YAML, Networking, Subprocess

## Run
pip install -r requirements.txt
python app.py
